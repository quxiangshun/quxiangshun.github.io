联系我们 ​

编辑日期：2024-10-14

感谢您对 Ly 系列开源项目的关注和支持！如果您有任何问题、建议或合作意向，欢迎通过以下方式与我们联系。

联系方式 ​

1. GitHub ​

您可以通过 GitHub Issues 与我们交流：

  • LyEdu：GitHub Issues
  • LyMom：GitHub Issues
  • LyDocs：GitHub Issues

2. Gitee ​

对于国内用户，您也可以通过 Gitee Issues 与我们交流：

  • LyEdu：Gitee Issues
  • LyMom：Gitee Issues
  • LyDocs：Gitee Issues

3. GitCode ​

  • LyEdu：GitCode Issues
  • LyMom：GitCode Issues
  • LyDocs：GitCode Issues

4. 邮件 ​

如果您有重要的事情需要联系我们，可以发送邮件到：

  • 技术支持：tech@ly-docs.com
  • 合作咨询：cooperation@ly-docs.com
  • 其他问题：contact@ly-docs.com

关于我们 ​

Ly 系列开源项目是由一群热爱开源的开发者共同维护的，我们致力于为企业和开发者提供高质量、易用的解决方案。

我们的愿景是通过开源技术，推动企业数字化转型，帮助企业提高效率、降低成本、提升竞争力。

如果您对我们的项目感兴趣，欢迎加入我们的社区，一起为开源事业贡献力量！
